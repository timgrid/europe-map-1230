# ADR-0003: Проекция Natural Earth I для отображения карты

Дата: 2025-01-01

## Статус

**Accepted**

## Контекст

GeoJSON из `aourednik/historical-basemaps` хранится в широте/долготе (EPSG:4326), а 3D-сцена
работает в плоских координатах XZ-плоскости. Нужна проекция, которая:

- минимизирует искажения для Европы (территория между 35°N-70°N, 25°W-60°E);
- дешева в вычислении (тысячи вершин на год);
- уже доступна в `d3-geo` (уже зависимость для типов);
- даёт приятный «глобусный» вид без артефактов Меркатора у полюсов.

## Решение

Использовать **d3.geoNaturalEarth1()** (Natural Earth I, Šavrič et al. 2015). Псевдоцилиндрическая
проекция с компромиссом между равновеликостью и компромиссной формой. После проекции инвертируем
ось Y (d3 использует SVG-конвенцию y-down, в 3D ожидаем север=+Z).

## Последствия

**Плюсы:** уже в `d3-geo` без новых зависимостей; приятные глазу формы; масштаб фиксирован
(`.scale(1)`), дальнейший масштаб делается умножением на `data.scale`. **Минусы:** проекция
не сохраняет расстояния и площади точно — для политической карты это допустимо, для навигации
нет. **Риски:** при желании показывать весь мир, а не Европу, придётся выбрать другую проекцию.
Для Европы + Ближний Восток + Северная Африка Natural Earth I хорошо подходит.
