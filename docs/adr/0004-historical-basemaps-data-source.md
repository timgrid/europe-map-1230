# ADR-0004: Источник данных — aourednik/historical-basemaps

Дата: 2025-01-01

## Статус

**Accepted**

## Контекст

Нужны исторические границы европейских государств за 1200-1530 годы. Требования:

- формат GeoJSON (нативная поддержка в three.js/d3-geo);
- лицензия, допускающая коммерческое использование и публикацию;
- покрытие нужных годов;
- разумный размер файлов (десятки мегабайт, а не гигабайты);
- единый стиль атрибуции стран между годами (стабильные id/name).

## Решение

Взять данные из репозитория **aourednik/historical-basemaps** (на GitHub, MIT-лицензия,
~1-4 МБ на год для Европы). Использовать `world_{year}.geojson` (мировой охват; в скрипте
препроцессинга обрезаем bbox Европы + ближайшие территории).

## Последствия

**Плюсы:** открытая лицензия MIT, единый источник для всех годов, проект поддерживается.
**Минусы:** названия стран меняются между годами (`England and Ireland` → `England`),
нужна нормализация в `commonAliases`. **Риски:** проект может прекратить обновление —
данные уже в git через processed JSON, можно мигрировать на альтернативу (например,
`wge/world-history-gazetteer`).

## Источники

- https://github.com/aourednik/historical-basemaps
- https://doi.org/10.1038/sdata.2018.25 (Aourcednik et al. 2018)
